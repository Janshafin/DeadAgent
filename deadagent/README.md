# DeadAgent Protocol

DeadAgent is a decentralized dead man's switch and AI agent succession protocol.

## Development Transparency

Initial scaffold committed as a single batch after architecture planning phase. All integration work committed incrementally thereafter.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
